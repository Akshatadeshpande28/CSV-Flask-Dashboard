# 📊 CSV Flask Dashboard

A simple Flask web application where users can upload a CSV file and view:

- Summary statistics
- Column overview
- Distribution charts

## 🚀 Features
- Upload CSV file
- Automatic data analysis using Pandas
- Visualization using Matplotlib

## 🛠 Tech Stack
- Python
- Flask
- Pandas
- Matplotlib

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
